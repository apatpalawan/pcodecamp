1. แท็ก DOCTYPE คืออะไร

คือ การประกาศว่าเอกสารเว็บเพจนั้นสร้างขึ้นมาโดยยึดหลักตามมาตราฐานใด ซึ่งมาตราฐานที่เราใช้กันทุกวันนี้กับการแสดงผลเว็บเพจ จะมีที่ใช้กันอยู่ 3 มาตราฐาน คือ
- HTML
- XHTML
- HTML 5

ทั้ง 3 มาตราฐาน จะมีการแบ่งกฎในการพัฒนาแตกต่างกันออกไปอีก 3 ประเภท คือ - Transitional
- Strict

3. แท็ก < b>, < i> ต่างจาก < strong>, < em> หรือไม่ อย่างไร?

< b> สำหรับการทำข้อความให้เห็นเป็น ตัวหนา หรือตัวเข้ม แต่จะใช้ < strong> ในการเน้นความสำคัญของข้อความ และ tag < em> เป็นการเพิ่มการเน้นย้ำให้กับข้อความที่แท็กนี้ครอบอยู่ ซึ่งจะให้การแสดงผลเหมือนกับแท็ก < i> ที่เราเคยๆใช้กันมา แต่จะต่างกันตรงที่แท็ก < i> แค่ทำให้ข้อความที่ครอบเป็นตัวเอียงเท่านั้นไม่มีการสื่อความหมายอื่นใดทั้งสิ้น
